## Replace the live site

1. Extract this ZIP.
2. In the existing GitHub `baja-232` repository, choose **Add file → Upload files**.
3. Upload everything inside this folder, including the `video`, `audio`, and `images` folders.
4. Allow GitHub to replace the existing files.
5. Commit the changes.
6. Vercel should redeploy automatically.

Because browsers normally block websites from autoplaying sound, the opening video begins muted. The separate Captain’s Call button plays the exhaust audio after the visitor clicks it.
